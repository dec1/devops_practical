- ##### TLS 
    - replaces **SSL** 
    - authentication and encryption uses
        - **certs** (X.509) - which include key pairs along with other information  

   - Relies on a **chain of trust** established by CAs.



- ##### Standard key locations



    - public (certs):
    `/etc/ssl/certs/`
    
    - private (keys):
    `/etc/ssl/private/`
