## Find files



- **`find`** **`-name file_name`**   `[startDir]` `[-i | -iname]` `[-type f|d|l] `Â      
    - `i` or `-iname`  case insensitive 
    - `startDir` curr dir if omitted
    - `type` search for (default= all) 
        - `f` files
        - `d` dirs
        - `l` links 
    - `file_name`   can contains **wildcards**
